Proyectos

Sistema gestion de articulos

Es un aplicacion echa en C#, .NET framework y WinForms para la asignatura Programacion II de la universidad UTN Gral Pacheco, el cual se encarga de gestionar los articulos de un catalogo de un comercio generico. Cuenta con un entorno grafico y menu muy intuitivo para el usuario. Al ser generico se puede utilizar con cualquier base de datos

Sitema de gestion para un torneo deportivo

Es un sistema de consola desarrollado con C++, que se encarga de cargar, eliminar y modificar los datos de los jugadores inscriptos, deportes a competir y los equipos inscriptos, para un torneo deportivo ficticio. Los reportes se almacenan en .txt. Este sistema fue desarrollado como trabajo practico en la asignatura Laboratorio II de UTN Gral Pacheco

Boss Rabbits Rabber

Juego de plataforma que se ejecuta en consola desarrollado en Java, en el cual el jugador toma el control de un conejo espacial cuyo objetivo es sobrevivir hasta que pase cierto tiempo. El juego cuenta con algunos obtaculos y mecanicas que nos impediran/ayudaran alcanzar el objetivo. Fue desarrollado como trabajo practico grupal para la asignatura Programacion I de UNGS.
